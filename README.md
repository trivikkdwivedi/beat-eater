# Double K | Visual Experience

This is the official cinematic homepage for music artist Double K (Trivikk Dubey).

## Tech Stack
- **Core:** HTML5, CSS3
- **Visuals:** WebGL (OGL Library) for Aurora Borealis effects
- **Audio:** Embedded Click SFX
- **Tooling:** Vite

## Visual Identity
- **Theme:** "Holy vs Sinful" / Dark Cinematic
- **Colors:** Matte Black (#050505), Deep Red (#C30000), Off-White
- **Typography:** Playfair Display (Serif) & Montserrat (Sans)

## Setup
1. Install dependencies: `npm install`
2. Start the studio: `npm run dev`
